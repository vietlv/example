Manual

Getting started

Once you log in, you'll be taken to the settings page. Choose a map and a layer that you want to populate with new data.

There's a sample map and a layer that you can use for testing purposes. Of course, later you can choose between the maps you've created in the GIS Cloud Map Editor and the maps someone has shared with you.

If you don't have a GIS Cloud account, sign up for one. It's free!

Offline mode

You can choose to work in offline mode and collect data even if your device doesn't have an internet connection available. Note that you can activate the offline mode at will by choosing Network access off in settings.

In offline mode you won't be able to log in, view the map, or switch between the maps and layers so make sure that you've set your project while online. All the data you collect while offline can be found in the queue. You can choose to delete data submissions there. Once you connect to the internet, the submissions you've collected will be immediately sent.

Cellular data connection

If you set the Send using cellular data network option to No, your device will send the data you collect only via Wi-fi connection and preventing unnecessary mobile internet traffic.

Collect Data

To collect data simply go to the Form, enter the required data and tap Send.

My location and data view

In the Map tab you can view all the layers that are in the map and review the attribute data of all the submissions simply by tapping on the point.

After a submission, tap Refresh to see the new point on the map.

Note that the map is unavailable in offline mode.

For more information on how to create maps, layers, custom forms etc. check out the User Manual on the GIS Cloud web page.
